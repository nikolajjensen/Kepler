\doxysubsection{Detailed Description}
Central data structure in Kepler.

Represents a multidimensional array of elements. The elements can be of either type String, Number or \mbox{\hyperlink{structkepler_1_1_array}{Array}}.

Elements can be of mixed type.

The shape of the array is stored in the shape vector, while the data is stored in the data vector.
