\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classkepler_1_1_symbol_table}{Symbol\+Table}} is used to map identifiers to their values.

A \mbox{\hyperlink{classkepler_1_1_symbol_table}{Symbol\+Table}} can have a parent \mbox{\hyperlink{classkepler_1_1_symbol_table}{Symbol\+Table}}, which is used to look up identifiers which are not defined in the current \mbox{\hyperlink{classkepler_1_1_symbol_table}{Symbol\+Table}}.
