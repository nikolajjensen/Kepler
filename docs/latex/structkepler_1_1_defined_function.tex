\doxysubsection{Detailed Description}
A Defined \mbox{\hyperlink{structkepler_1_1_function}{Function}} contains the \mbox{\hyperlink{structkepler_1_1_anonymous_function}{Anonymous\+Function}} \mbox{\hyperlink{structkepler_1_1_a_s_t_node}{ASTNode}} (and subtree) which pertains to a specific user-\/defined function. The \mbox{\hyperlink{structkepler_1_1_defined_function}{Defined\+Function}} \mbox{\hyperlink{structkepler_1_1_operation}{Operation}} is applied to the arguments in a similar way to primitive functions.

Concretely, the \mbox{\hyperlink{structkepler_1_1_defined_function}{Defined\+Function}} will create a new \mbox{\hyperlink{structkepler_1_1_interpreter}{Interpreter}} object and use it to evaluate the function body with the given arguments.
