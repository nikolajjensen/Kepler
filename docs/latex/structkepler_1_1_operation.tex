\doxysubsection{Detailed Description}
Represents an arbitrary \mbox{\hyperlink{structkepler_1_1_operation}{Operation}} to be applied to some data.

Concrete operations can inherit from this class, and implement whichever of the operator() methods they need. In this way, any arguments they do not support will be handled by the default implementation, which throws a Domain\+Error.
